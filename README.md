# Calculator Web App

A responsive calculator built using HTML, CSS, and JavaScript.  
This project was created to practice DOM manipulation, event handling, and basic arithmetic logic in JavaScript.

---

## Features

- Basic arithmetic operations (+, −, ×, ÷)
- Clear (C) button to reset input
- Delete (⌫) button to remove last digit
- Responsive layout
- Hover effects for better UI
- Error handling (e.g., divide by zero)

---

## Tech Stack

- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript (Vanilla JS)

---

## Concepts Practiced

- DOM Selection & Manipulation
- Event Listeners
- Functions
- Conditional Logic
- String Handling
- Responsive Design

---

## Learning Objective

The goal of this project was to:

- Understand how JavaScript interacts with HTML elements
- Build logical thinking for arithmetic operations
- Improve frontend structure and styling skills
- Practice writing clean, maintainable code
- Keyboard input support
- Scientific calculator functions
- Theme toggle (Dark/Light mode)
- Improved UI animations

---

## Responsive Design

The calculator is fully responsive and works on:
- Mobile devices
- Tablets
- Desktop screens
